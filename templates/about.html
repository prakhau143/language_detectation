How It Works

1

Unicode Analysis

Detects Devanagari characters (\u0900-\u097F) for Hindi text identification with strong signal weighting.

2

Token Analysis

Uses English dictionary and stopwords to identify English text patterns and filter false positives.

3

Pattern Matching

Identifies Hinglish using Roman Hindi verbs, particles, and bigram patterns like "ja raha", "kar raha".

4

Confidence Scoring

Calculates weighted scores for each language and returns the highest confidence result with percentage breakdown.
